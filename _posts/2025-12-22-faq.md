## FAQ
在多个版本vs（问题机器是vs2022）并存的系统上，编译64位程序时报错 
```
C:\Program Files (x86)\Windows Kits\10\Include\10.0.19041.0\um\winnt.h(34,10): fatal error C1083: 无法打开包括文件: “ctype.h”: No such file or directory
```
解决方案是：
```
HKLM\SOFTWARE\Microsoft\Windows Kits\Installed Roots                          64位的sdk查找根目录
HKLM\SOFTWARE\WOW6432Node\Microsoft\Windows Kits\Installed Roots              32位的sdk查找根目录

正常情况下，这个里面的值会是C:\Program Files (x86)\Windows Kits\10\，所以检查下这俩是不是都是这个，不是修改成这个就可以了。
```
